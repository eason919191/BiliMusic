# BiliMusic 🎵

基于哔哩哔哩的音乐播放器 | Material Design 3 | Jetpack Compose

## 功能

- 🔍搜索 — B站视频搜索，支持排序（综合/播放量/收藏/弹幕/日期），过滤超长视频
- 📋 歌单 — 创建歌单、导入B站收藏夹、封面自动设置
- 🎤 歌词 — 基于B站AI字幕，实时滚动匹配
- 🔐 登录 — 扫码（B站App跳转）、网页登录（支持短信验证码）
- 🎛️ 系统播放控制 — 锁屏/通知栏/蓝牙耳机控制
- ⚙️ 设置 — 主题色、模糊强度、进度条样式、定时关闭、搜索筛选、排序
- 🌙 深色模式 — Material You 动态取色

## 技术栈

Kotlin · Jetpack Compose · Material 3 · Hilt · Room · DataStore · Retrofit · OkHttp · Media3 · Coil
