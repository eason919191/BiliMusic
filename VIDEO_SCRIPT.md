# BiliMusic 介绍视频脚本

> 预计时长：2-3分钟 | 风格：科技感 + 轻快节奏

---

## 开场 (0-10秒) — 展示App图标/启动画面

"你有没有想过——把哔哩哔哩变成一个纯粹的音乐播放器？

不是直接看视频，而是像网易云、QQ音乐那样，搜索、收藏、下载、歌词、后台播放……

这就是 BiliMusic。"

## 搜索与播放 (10-40秒)

【屏幕录制：搜索界面 → 输入关键词 → 结果展示 → 点击播放 → 迷你播放器弹出 → 全屏播放器】

"打开就是搜索。输入歌名、UP主、甚至视频描述，哔哩哔哩的海量资源都能搜到。

点一下直接播放——只有音频，不需要加载视频画面，省流量速度还快。

切到全屏播放器，视频封面高斯模糊做背景，Material Design 3的动态主题色，Pixel风格的进度条——视觉上就很舒服。"

## 歌词功能 (40-55秒)

【屏幕录制：播放中点击歌词按钮 → 歌词滚动显示】

"重点讲一下歌词——它是哔哩哔哩的**AI字幕**直接转过来的。

只要哔哩哔哩视频有AI字幕，BiliMusic就能把它当歌词用，还会根据播放进度自动滚动。

你知道哔哩哔哩有多少视频有AI字幕吗？知识区、纪录片、连很多音乐MV都有——这些都是你的歌词库。"

## 歌单与收藏夹 (55-1分20秒)

【屏幕录制：创建歌单 → 从哔哩哔哩导入收藏夹 → 歌单封面自动显示】

"歌单可以自己建，也可以直接从哔哩哔哩收藏夹一键导入。

如果你在哔哩哔哩收藏了几百个视频，BiliMusic能全部转成歌单，封面自动取第一首歌的图。

每首歌都有播放、下载、更多操作。长按还能批量选择，复制到其他歌单。"

## 登录 (1分20秒 - 1分35秒)

"登录也很聪明——扫码、哔哩哔哩App一键跳转，或者网页登录。

网页登录这个特别实用——它会打开哔哩哔哩官方页面，你正常输手机号、接收短信、完成验证，App自动提取cookie。哔哩哔哩的验证码系统全帮你挡掉。"

## 系统播放控制 (1分35秒 - 1分45秒)

【屏幕录制：锁屏状态 → 显示播放控制】

"播放的时候支持系统级的媒体控制。

锁屏上显示封面和歌曲信息，通知栏可以切歌暂停，连蓝牙耳机和车载都能控制。

后台播放，熄屏继续听，就像原生音乐App一样。"

## 下载 (1分45秒 - 1分55秒)

【屏幕录制：下载列表 → 显示速度、进度】

"下载用的是DASH音频流——不是整个视频，是纯音频。

一个视频几百兆，音频才三五兆。队列自动开始，后台多线程，速度和进度实时显示。"

## 设置 (1分55秒 - 2分05秒)

【屏幕录制：设置页面 → 滑动浏览各项设置】

"设置也很全。主题色、深色模式、模糊强度、进度条样式、定时关闭、

搜索还能按播放量排序，自动过滤十分钟以上的视频，或者标题带'循环'的视频——用哔哩哔哩搜音乐的老大难问题，都帮你解决。"

## 结尾 (2分05秒 - 2分15秒)

【屏幕录制：回到主界面 → 展示App名 → 画面定格】

"BiliMusic，开源在GitHub。

Kotlin + Jetpack Compose，Material Design 3，AI辅助开发，一个人肝出来的。

链接在简介里，欢迎Star和PR。"

## 录制建议

| 场景 | 操作 |
|------|------|
| 屏幕录制 | 手机开启USB调试，用 `scrcpy` 投屏到电脑录制 |
| 配音 | 用剪映或CapCut的TTS（文本转语音），选"阳光男声"或"知性女声" |
| 字幕 | 剪映自动识别字幕，手动调整关键帧 |
| 背景音乐 | 选一首轻电子/Chillwave，音量调低到20% |
| 分辨率 | 1080x1920（竖屏），适合哔哩哔哩发布 |
